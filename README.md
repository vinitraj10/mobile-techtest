# Mobile tech test

### Running this project

Download and install all necessary packages:

    npm install

Install pods:

    npm podinstall

Start the server:

    npm start

Build on your open iOS simulator:

    npm run ios

Build on your open android simulator:

    npm run android

### Testing and linting

Check your code with eslint:

    npm run lint

Autoformat your code using prettier:

    npm run format

Run your jest tests:

    npm run test
